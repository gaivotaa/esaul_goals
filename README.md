# ESAUL. IT проєкти Логістики — Цілі та задачі

Інтерактивний веб-дашборд для візуалізації, пошуку та фільтрації 34 ключових цілей ІТ-проєктів логістики компанії Kernel.

Проєкт виконано у преміальному стилі згідно з офіційним брендбуком компанії Kernel (кольорова палітра темно-зеленого, світло-зеленого, кремового та нейтральних відтінків, шрифт Montserrat).

## 🚀 Основні можливості

- **Групування за статусом:** Цілі автоматично розділені на дві групи: **«Включені в план»** та **«Беклог»** (з лічильниками кількості задач).
- **Два режими відображення:**
  - **Картки (Grid View):** Стильні картки зі скляним ефектом (glassmorphism) та мікроанімаціями.
  - **Таблиця (List View):** Адаптивна інтерактивна таблиця з основними тегами.
- **Швидкий пошук та розумна фільтрація:** Пошук за назвою/описом/замовником, а також фільтри за Системами (NAV, TransitHub), Статусом, Замовником, Напрямком бізнесу, рівнем Цінності та Складності.
- **Панель деталей (Detail Drawer):** Плавна бокова панель з порівняльними таблицями AS-IS / TO-BE, бізнес-цінністю та очікуваними ефектами.

## 📂 Структура проєкту

- `index.html` — основна розмітка та каркас дашборду.
- `style.css` — стилі дизайн-системи, анімації та адаптивна верстка.
- `app.js` — клієнтська логіка фільтрації, перемикання режимів та Drawer.
- `goals.json` — структурована база даних цілей (результат парсингу).
- `parse_backlog.py` — Python-парсер, який конвертує вихідний markdown беклогу у формат JSON.
- `.gitignore` — виключення технічних файлів з репозиторію.

## 🛠️ Як запустити локально

1. Склонуйте репозиторій або завантажте файли.
2. Перейдіть до папки проєкту та запустіть локальний HTTP-сервер:
   ```bash
   python -m http.server 8000
   ```
3. Відкрийте в браузері сторінку:
   [http://localhost:8000/index.html](http://localhost:8000/index.html)
